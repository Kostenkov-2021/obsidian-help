Если вы разработчик, вы можете внести вклад в развитие Obsidian различными способами.

### Плагины и темы оформления

Вы можете разрабатывать [[Пользовательские плагины]] и [[Темы оформления]], используя TypeScript и CSS. Узнайте больше, перейдя в [Документацию для разработчиков Obsidian](https://docs.obsidian.md).

### Проекты с открытым исходным кодом

Obsidian разрабатывает и поддерживает несколько проектов с открытым исходным кодом. Вы можете отправлять pull requests с исправлениями ошибок и улучшениями в следующие репозитории GitHub:

*   [JSON Canvas](https://github.com/obsidianmd/jsoncanvas) (MIT) — открытый формат файлов для данных бесконечного холста.
*   [Obsidian Importer](https://github.com/obsidianmd/obsidian-importer) (MIT) — плагин, который конвертирует данные из множества различных форматов в простые текстовые файлы Markdown.
*   [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper) (MIT) — браузерное расширение для выделения и сохранения веб-страниц в виде простых текстовых файлов Markdown.
*   [Defuddle](https://github.com/kepano/defuddle) (MIT) — парсер HTML-to-Markdown, используемый Obsidian Web Clipper.
*   [Obsidian Maps](https://github.com/obsidianmd/obsidian-maps) (MIT) — плагин, который отображает карты в основах Obsidian.

### Документация

Чтобы помочь улучшить нашу [документацию для разработчиков](https://docs.obsidian.md/Home), отправьте pull request в репозиторий [obsidian-developer-docs](https://github.com/obsidianmd/obsidian-developer-docs).

### Сообщество

Канал [plugin-dev](https://discord.com/channels/686053708261228577/840286264964022302) на нашем официальном сервере Discord — это место, где вы можете обсуждать разработку плагинов и помогать другим разработчикам с их плагинами.
